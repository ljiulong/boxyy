# Boxy TUI

Boxy TUI provides a terminal UI for package managers. It shares core logic with the CLI.

## Run

```bash
cargo run -p boxy-tui
```

## Controls

- j/k or Up/Down: Move selection
- h/l or Left/Right: Switch manager (Dashboard)
- Enter: Open package detail
- /: Search
- u: Update package
- d: Uninstall package
- r: Refresh data
- L: Show job logs
- b/Esc: Back
- q: Quit
- Ctrl+C: Quit
- ?: Toggle help
