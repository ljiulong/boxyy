# Boxy GUI (Tauri)

This crate provides the Tauri backend for the Boxy GUI.

## Run (dev)

```bash
cd ../gui-frontend
pnpm install
pnpm dev
```

In another terminal:

```bash
cd ../boxy-gui
cargo run -p boxy-gui
```

## Tauri dev (recommended)

```bash
cd ../gui-frontend
pnpm install
pnpm tauri:dev
```
